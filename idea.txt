### sendgrid API key Steps
1. Create an Account: Go to SendGrid.com and sign up for a free account.
2. Verify Sender Identity:
    - Go to Settings > Sender Authentication (https://app.sendgrid.com/settings/sender_auth/senders/new).
    - Click Verify a Single Sender.
    - Fill in your details (use the same email you put in EMAIL_USER in your .env).
    - Verify your email by clicking the link sent to your inbox.
3. Create API Key:
    - Go to Settings > API Keys (https://app.sendgrid.com/settings/api_keys). 
    - Click Create API Key.
    - Give it a name (e.g., "My App").
    - Select Full Access (easiest) or Restricted Access (select "Mail Send" permissions).
    - Click Create & View.
4. Copy the Key:
    - IMPORTANT: Copy the key immediately (it starts with SG.). You won't see it again.
    - Paste it into your .env file as SENDGRID_API_KEY=SG.xxxxxxxx....
