Sample Project

This is a sample project to demonstrate how to add new projects to the projects directory.

To add a new project:

  1. Create a new folder in the projects directory (e.g., my-new-project).
  2. Add an index.html file inside that folder.
  3. Add any assets (CSS, JS, images) in the same folder or a public subfolder.
  4. Access it at /projects/my-new-project/.
